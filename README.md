# cursive-script

# 草書大師

Font used: [衡山毛筆フォント草書](https://opentype.jp/kouzansousho.htm)

| Key | Description |
| ------ | ----------- |
| F1 | help |
| Ctrl + S   | switch between cursive script and default font. |
| PageUp | make font size bigger. |
| PageDown | make font size smaller. |
